# Task: Insurance Data Visualization Application

Create a **ReactJS** application for Insurance Data Visualization using **Node.js** and **MySQL**.

## Main Features

The application should have **3 main tabs**:

### 1. Database Engine
- Create/Delete tables
- CRUD operations
- Upload data via Excel (.xlsx/.csv)
- View and manage data

### 2. Dashboard Engine
- Create dashboard views (charts, summary tables, KPIs)
- Setup filters (e.g., date, region, policy type)
- Save/manage dashboard configurations
- Efficient view creation like Qlik Sense

### 3. Dashboard View
- Select dashboard
- Apply filters dynamically
- View updated visualizations
- Export dashboard (PDF/image)

## Optional Enhancements
- ✅ Authentication
- ✅ Role-based access
- ⏳ Responsive design
- ✅ Use of chart libraries (Chart.js, Recharts, etc.)

## Deliverables
- ✅ Source code
- ✅ Setup instructions
- ✅ Sample data
